## 2024-05-15 - Insecure JSON Deserialization in Godot User Files
**Vulnerability:** Reading user-controlled files (`user://...`) like save files or settings using Godot's `JSON.new().parse()` without strictly validating the resulting `json.data` type.
**Learning:** Godot's `JSON.parse` allows strings, numbers, or arrays to be returned as `json.data`. If the code blindly casts this to a Dictionary (e.g. `var data: Dictionary = json.data`), it can crash the application when a modified/malformed file is loaded, causing a localized Denial of Service.
**Prevention:** Always validate parsed JSON with `typeof(json.data) == TYPE_DICTIONARY` before using dictionary methods or assigning to a strongly-typed Dictionary variable.
